Privacy Policy — Awon

Last updated: July 17, 2026 · Contact: joshuamerlene@gmail.com

Awon is a social media management application. It connects to platforms such as TikTok through their official APIs, on behalf of a user who has authorized the connection, to help that user prepare, schedule, and publish their own content. This policy explains what data Awon accesses, how it is used, and how it is protected.

What data we access

  • Connected account data: when a user connects an account (e.g., TikTok) via OAuth, Awon receives an access token and, through that platform's API, basic profile information (display name, username, avatar), available posting options (privacy levels, interaction settings), and performance statistics for videos on the connected account.
  • Content the user provides: raw footage or media the user uploads for processing, and the resulting edited videos, captions, and metadata prepared from that content.
  • Awon does not access direct messages, contacts, or data belonging to any user other than the one who authorized the connection.

How we use it

  • To display accurate posting options (privacy levels, interaction settings) before any post is made.
  • To publish content to a connected account — only content the user has provided or approved, and only with that user's authorization.
  • To show the user activity logs and performance statistics for their own published content.

What we do NOT do

  • We do not sell, rent, or share account data with any third party.
  • We do not use connected-platform data for advertising, profiling, or any purpose other than operating the account the user has authorized.
  • We do not post content without authorization from the account owner.
  • We do not knowingly collect data from children under 13. The service is intended for adult users.

Storage and security

  • Access tokens and content are stored on a private, access-controlled server and are not exposed publicly.
  • API credentials and secrets are stored as environment variables, never in source code or public repositories.
  • Uploaded footage and edited clips are retained only as long as needed for content production; edited clips are deleted after posting or discarding.

Data retention and deletion

A user can disconnect a connected account at any time by revoking Awon's access in that platform's own settings (for TikTok: Settings & privacy → Security & permissions → Apps and services), which invalidates the stored token. Uploaded footage and derived content can be deleted at any time on request to joshuamerlene@gmail.com; requests are honored within 30 days.
